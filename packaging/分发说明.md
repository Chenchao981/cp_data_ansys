# CP数据分析工具 - 分发说明

## 📦 分发内容

将 `dist` 文件夹整个打包分发给用户即可。

### 📁 dist 文件夹包含

```
dist/
├── 📄 requirements.txt         # Python依赖清单
├── 🚀 start.bat                # 程序启动脚本
├── 💾 install_offline.bat      # 离线安装依赖脚本
├── 📦 app.pyz                  # 主程序包
├── 📁 wheels/                  # 离线依赖包（17个文件，约35MB）
└── 📖 用户安装使用说明.md       # 用户使用指南
```

## 🎯 用户使用流程

1. **解压文件** - 用户收到文件后解压
2. **安装Python** - 确保已安装Python 3.8+
3. **安装依赖** - 选择在线或离线安装：
   - 在线：`pip install -r requirements.txt`
   - 离线：双击 `install_offline.bat`
4. **启动程序** - 双击 `start.bat`

## ✅ 功能特点

- ✅ **支持离线安装** - 无需网络连接
- ✅ **一键安装** - 用户操作简单
- ✅ **自动检测** - 自动检查环境
- ✅ **中文界面** - 友好的中文提示
